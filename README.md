# SalesFlow: Furniture Retail Sales Analysis

## Project Overview
This project focuses on analyzing sales data for a furniture store to identify top-performing categories, countries and seasonal trends.

## Tools Used
- **Python Libraries:** Pandas, NumPy, Matplotlib, Seaborn (visualization), SciPy/Statsmodels (statistical testing)
** Data cleaning and Exploratory Data Analysis (EDA).
- **Tableau:** Interactive dashboarding and storytelling.
